# Решение а) методом for:

```
# Задаем исходный массив
arr = [1, 2, 3, 4, 5]

# Увеличиваем все элементы в 2 раза циклом for
for i in range(len(arr)):
    arr[i] *= 2

# Выводим полученный массив
print(arr)
```

# Решение б) методом comprehension:

```
# Задаем исходный массив
arr = [1, 2, 3, 4, 5]

# Уменьшаем все элементы на число А с помощью comprehension
a = 3
new_arr = [x - a for x in arr]

# Выводим полученный массив
print(new_arr)
```

# Решение в) методом map:

```
# Задаем исходный массив
arr = [1, 2, 3, 4, 5]

# Делим все элементы на первый элемент с помощью функции map
new_arr = list(map(lambda x: x / arr[0], arr))

# Выводим полученный массив
print(new_arr)
